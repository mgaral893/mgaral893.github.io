| [HandsOff: Anti-Theft Alarm](https://mgaral893.github.io/android/handsoff/) | `com.safephone.antitheft` | [Privacy](https://mgaral893.github.io/android/handsoff/privacy/) | [Landing](https://mgaral893.github.io/android/handsoff/) |

## Adding a new app

1. Copy the [privacy template](../templates/privacy/index.html) to `android/<app-name>/privacy/index.html` and fill in the placeholders.
2. Create `android/<app-name>/index.html` (use the HandsOff landing as reference).
3. Add a row to the table above.
4. Commit and push to `main` — GitHub Pages deploys automatically.
5. Set the Play Console `contactWebsite` for the new app to `https://mgaral893.github.io/` (the root).
