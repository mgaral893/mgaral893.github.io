Privacy Policy Template

Reusable template for Android apps published by mgaral893. Copy this file to /android/<app-name>/privacy/index.html and fill in the placeholders.

Placeholders to replace

  • {{APP_NAME}} — full app title (e.g. "HandsOff: Anti-Theft Alarm")
  • {{APP_PACKAGE}} — Android package id (e.g. "com.safephone.antitheft")
  • {{APP_SHORT_NAME}} — short identifier for the URL path (e.g. "handsoff")
  • {{LAST_UPDATED}} — date of last update
  • {{DATA_DESCRIPTION}} — what data the app actually processes
  • {{THIRD_PARTY_SDKS}} — any third-party SDKs in use

Template HTML

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>{{APP_NAME}} Privacy Policy</title>
  <link rel="canonical" href="https://mgaral893.github.io/android/{{APP_SHORT_NAME}}/privacy/">
  <!-- copy CSS from existing privacy pages for visual consistency -->
</head>
<body>
  <main>
    <p><a href="/">← Developer website</a></p>
    <h1>Privacy Policy — {{APP_NAME}}</h1>
    <p>Package: <code>{{APP_PACKAGE}}</code><br>Last updated: {{LAST_UPDATED}}</p>

    <h2>1. Data processed by the app</h2>
    <p>{{DATA_DESCRIPTION}}</p>

    <h2>2. Third-party services</h2>
    <p>{{THIRD_PARTY_SDKS}}</p>

    <h2>3. Data retention and removal</h2>
    <p>Data is stored only on the device and removed when the app is uninstalled.</p>

    <h2>4. Contact</h2>
    <p>For privacy questions, contact <a href="mailto:mgaral893@gmail.com">mgaral893@gmail.com</a>.</p>
  </main>
</body>
</html>
