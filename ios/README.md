# iOS apps

Reserved for future iOS apps published by mgaral893.

No apps yet. See [`/android/`](../android/) for the current app catalog (Android-first).

When an iOS app is added:
1. Create `ios/<app-name>/` with `index.html` (landing) and `privacy/index.html`.
2. Add a row to the table at `/`.
3. The App Store Connect `Marketing URL` should be set to `https://mgaral893.github.io/ios/<app-name>/` (not the privacy sub-path; App Store Connect has a separate field for `Privacy Policy URL`).
